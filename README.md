# libNN
#### A (deep) neural network library built from scratch in Python 3

### Network
1. Fully-connected feedforward neural network


### Weight Initialization

3. ReLU

### Optimizer
1. SGD
2. Adagrad

4. RMSprop
5. Adam


### Regularizer
1. L2 regularizer


### Others
